# Movie-Recommendation-System
Movie Recommendation based on collaborative filtering 
This movie recommendation system is based on collabrative filtering and using K-Nearest Neighbor.
The hamming distance is used as the metric to measure the dependence of one user to another.

##-------------##
The movie rating data used is downloaded from MovieLens|GroupLens (http://files.grouplens.org/datasets/movielens/ml-latest-small.zip)
